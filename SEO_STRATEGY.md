# SEO Strategy for vnmarketinsights.com

## Executive Summary

This SEO strategy positions vnmarketinsights.com as a neutral, authoritative reference destination for Vietnam market information. The approach prioritizes topical authority, trust signals, and long-term organic visibility over short-term traffic spikes.

## 3. CONTENT OPTIMIZATION GUIDELINES

### First 150 Words Template

Every article must begin with a clear definitional paragraph:

```
[TOPIC] in Vietnam refers to [DEFINITION]. As of [YEAR],
the market represents approximately [SIZE] with [KEY STATISTIC].
This [CATEGORY] encompasses [COMPONENTS] and serves [PURPOSE/FUNCTION].
```

Example (Digital Economy):
```
Vietnam's digital economy encompasses all economic activity derived from
digital technologies, platforms, and services. As of 2024, the sector
generates approximately $23 billion in annual value, representing 5.5%
of national GDP. This economy spans e-commerce, fintech, digital content,
online travel, and digital services, supported by 77 million internet users
and 72 million smartphone users.
```

### Entity SEO Requirements

#### Companies to Mention (Where Relevant)
- **Banks**: Vietcombank, BIDV, VietinBank, Techcombank, VPBank, ACB
- **E-wallets**: MoMo, ZaloPay, ViettelPay, ShopeePay, VNPay
- **E-commerce**: Shopee, Lazada, Tiki, TikTok Shop
- **Logistics**: Vietnam Post, GHN, GHTK, Viettel Post
- **Platforms**: Grab, Zalo, VNG, Viettel

#### Institutions to Reference
- State Bank of Vietnam (SBV)
- Ministry of Industry and Trade (MOIT)
- Ministry of Information and Communications (MIC)
- Vietnam E-commerce Association (VECOM)
- National Payment Corporation of Vietnam (NAPAS)

#### Regulatory References
- Decree 52/2013/ND-CP (E-commerce)
- Circular 19/2016/TT-NHNN (Payment services)
- Personal Data Protection Decree (2023)
- Cybersecurity Law (2018)

### Temporal Neutrality Guidelines

| Avoid | Use Instead |
|-------|-------------|
| "Recently" | "As of [YEAR]" |
| "Currently" | "In [YEAR]" or omit |
| "Just launched" | "Launched in [DATE]" |
| "Growing rapidly" | "Growing at [X]% annually" |
| "New trend" | "Emerging since [YEAR]" |

### List and Table Usage

Favor structured formats:
- Market share breakdowns: Use tables
- Platform comparisons: Use tables
- Step-by-step processes: Use numbered lists
- Component breakdowns: Use bullet lists

### Link Placement Logic

- Link to parent/hub page
- Link to closely related sibling page

- Link to specialized sub-topics

#### FAQ Schema (Select Pages Only)
Only implement where naturally occurring FAQ sections exist:
```json
{
  "@context": "https://schema.org",
```

### Schema Implementation Notes

- Use `Article` for all content pages
- Use `Dataset` for pages with substantial statistical content
- Use `BreadcrumbList` on all pages
- Use `FAQPage` only when genuine FAQ content exists
- Do NOT use `Review`, `AggregateRating`, or author reputation markup
- Include `dateModified` for content updates
- Use `Organization` markup consistently

## 6. TECHNICAL SEO (ASTRO-SPECIFIC)

### Sitemap Configuration

```javascript
// astro.config.mjs
import sitemap from '@astrojs/sitemap';

export default defineConfig({
  site: 'https://vnmarketinsights.com',
  integrations: [
    sitemap({
      changefreq: 'monthly',
      priority: 0.7,
      lastmod: new Date(),
      filter: (page) => !page.includes('/admin') && !page.includes('/draft'),
    })
  ]
});
```

### Canonical URL Implementation

```astro
---
// In each .astro page file
const canonicalURL = new URL(Astro.url.pathname, 'https://vnmarketinsights.com');
---

<head>
  <link rel="canonical" href={canonicalURL} />
</head>
```

### Clean HTML Output

- Use semantic HTML (`article`, `section`, `header`, `footer`)
- Proper heading hierarchy (H1 → H2 → H3)
- No inline styles in content
- Minimal inline JavaScript
- Alt text for all images

### Markdown-First Indexing

```astro
// Content collection approach
import { getCollection } from 'astro:content';

export async function getStaticPaths() {
  const articles = await getCollection('articles');
  return articles.map(article => ({
    params: { slug: article.slug },
    props: { article },
  }));
}
```

### Performance Optimization

```javascript
// astro.config.mjs
export default defineConfig({
  build: {
    format: 'file',  // Static HTML files
  },
  vite: {
    build: {
      cssCodeSplit: true,
      rollupOptions: {
        output: {
          manualChunks: {
            search: ['./src/components/Search.astro']
          }
        }
      }
    }
  }
});
```

### Robots.txt Configuration

```
User-agent: *
Allow: /

```

- Cite official statistics
- Reference academic research
- Link to government publications
